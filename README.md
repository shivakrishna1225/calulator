# Simple Calculator

This is a simple calculator application built using `Tkinter`, Python's built-in GUI toolkit. It allows users to perform basic arithmetic operations like addition, subtraction, multiplication, and division through an easy-to-use graphical interface.

## Features

- **Addition (+)**, **Subtraction (-)**, **Multiplication (*)**, and **Division (/)** operations.
- **Clear (C)** button to reset the input field.
- **Equals (=)** button to evaluate the current mathematical expression.
- **Intuitive layout** with buttons for digits (0-9) and operations.

## Prerequisites

To run the calculator, you need to have Python installed on your machine. This project uses the `Tkinter` module, which is part of the Python Standard Library, so no additional installations are needed.

- Python 3.x

## How to Run

1. **Clone or Download** this repository.

2. **Run the Python file**:
   
   Open a terminal or command prompt in the project directory and run the following command:

   ```bash
   python calculator.py
3.The calculator window will appear, and you can start performing calculations.
